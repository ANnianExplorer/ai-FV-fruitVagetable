## 福禄蔬菜[果蔬]店

### 项目介绍
本项目（福禄蔬菜（果蔬）店）是专门为餐饮企业（餐厅、饭店），当地居民定制的一款软件产品，包括系统管理后台和移动端应用两部分。
其中系统管理后台主要提供给蔬菜店企业内部员工使用，可以对果蔬的分类、果蔬品种、组合、订单、员工、通知、优惠券等进行管理维护。
移动端应用主要提供给消费者使用，可以在线浏览果蔬、查看最新消息、查看个人信息、添加地址、使用优惠券、添加购物车、下单以及查看订单等。

- 后台管理：http://localhost:8081/backend/index.html
- 前端页面：http://localhost:8081/front/index.html
